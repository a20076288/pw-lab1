    HTML5 e CSS

    O HTML5 (Hypertext Markup Language) é a mais nova versão do padrão HTML. O termo representa dois conceitos diferentes. Esta é uma nova versão da linguagem HTML, com novos elementos, atributos e comportamentos, e um conjunto maior de tecnologias que permite a criação de sites e aplicações da Web mais diversos e poderosos.

    Elementos HTML Utilizado Descrição
    ‹ H1/H2/H3 ›done_allUsado nos cabeçalhos das páginas
    ‹ p ›done_allUsado para escrever parágrafos
    ‹ a ›done_allUsado para referenciar hiperlinks
    ‹ center ›dangerousDeixou de ser usado em HTML5
    ‹ hr ›done_allUsado para inserir uma barra horizontal de separação de conteúdos
    ‹ br ›dangerousCaiu em desuso no HTML5, é preferível usar a propriedade "margin"
    ‹ th ›done_allUsado para inserir os cabeçalhos nas tabelas
    ‹ tr ›done_allUsado para inserir as linhas na tabela
    ‹ td ›done_allUsado para inserir os dados nas células da tabela
    ‹ ul ›done_allCriado para usar uma lista não ordenada
    ‹ li ›done_allCriado para usar uma lista ordenada
    ‹ form ›done_allUsado para criar formulários
    ‹ input ›done_allUsado para registar inputs dos utilizadores
    ‹ img ›done_allUtilizado para inserir imagens
    ‹ iframe ›done_allUsado para inserir conteúdos numa única janela

    O CSS (Cascading Style Sheets) é o código usado para dar estilo ao website. O CSS descreve como elementos são mostrados no ecrã a partir do HTML já criado.

    Selectores CSS Utilizado Funcionamento
    *{Universal}done_allO seletor universal aplica o estilo a todos os elementos da página. Foi aplicado para selecionar o tipo de fonte a usar em todo o website
    body {Tipo}done_allO seletor de tipo usa o nome do marcador a estilizar. Foi aplicado para estilizar o body, headers, section, entre outros.
    .importante {Classe}done_allO seletor de classe associa um nome a um estilo. Foi usado para estilizar alguns elementos desta tabela, para dispor em "blocos" os campos da página de comentários, entre outros.
    #unico {Id}done_allO seletor de id permite identificar univocamente um elemento. Foi aplicado para definir que toda o texto e links da secção "Estrutura do Website" ficaria alinhado à esquerda em vez de centrado, entre outros.
    a[target=_blank] {Atributo}done_allSeletores que se aplicam a atributos de elementos. Foi aplicado para definir uma cor específica para todos os links externos a este website.
    a:visited {pseudo-classe}done_allSeletor associado ao estado de um elemento. Foi utilizado para customizar a cor e decoração dos links nos seus vários estados (clicado, visitado, selecionado).
    p::first-line {pseudo-elemento}done_allSeletor que se aplica a partes de elementos. Foi aplicado para definir que a primeira letra e o primeiro parágrafo do texto introdutório ficariam com cores diferentes do resto do texto
    Combinadores de seletores CSS Utilizado Descrição
    div pdone_allSeletores descendentes são separados por espaço. Foi utilizado para customizar o estilo de fonte no poema
    div > pdone_allSeletores filhos através do uso de >. Foi utilizado para selecionar a cor no texto do poema através de "divs".
    div ~ pdone_allSeletores adjacentes através do uso de ~. Foi utilizado para selecionar outra cor no texto do poema através de "divs".
    div + pdone_allSeletores imediatamente adjacentes através do uso de +. Foi usado para "sublinhar" alguns parágrafos no poema
    div, pdone_allAgrupar os seletores, separando-os por virgulas. Foi utilizado para colocar todos os divs e parágrafos, dentro do poema, em itálico
